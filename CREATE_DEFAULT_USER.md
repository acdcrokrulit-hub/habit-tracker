# Как создать пользователя mukhanovr@icloud.com в Supabase

Чтобы вы всегда могли заходить под аккаунтом `mukhanovr@icloud.com` с паролем `mukhanov06`, выполните следующие шаги:

## Способ 1: Через Supabase Dashboard (рекомендуется)

### Шаг 1: Откройте Supabase Dashboard
1. Перейдите на [https://supabase.com](https://supabase.com)
2. Войдите в свой аккаунт
3. Выберите ваш проект habit-tracker

### Шаг 2: Создайте пользователя
1. В левом меню нажмите **Authentication**
2. Выберите **Users**
3. Нажмите кнопку **Add user** (или **Invite user**)
4. Введите:
   - **Email**: `mukhanovr@icloud.com`
   - **Password**: `mukhanov06`
   - **Auto Confirm User**: ✅ **Включите** (это важно!)
5. Нажмите **Add user**

### Шаг 3: Проверьте
Теперь вы можете войти в приложении с этими данными.

## Способ 3: Зарегистрироваться через приложение

1. **Сначала отключите подтверждение email** (как описано в `DISABLE_EMAIL_CONFIRMATION.md`)
2. Запустите приложение
3. Нажмите "Создать аккаунт"
4. Введите:
   - Email: `mukhanovr@icloud.com`
   - Пароль: `mukhanov06`
5. Нажмите "Зарегистрироваться"
6. Так как подтверждение отключено, вы сразу войдете

## Важно!

- Если пользователь с таким email уже существует, вы получите ошибку "Пользователь с таким email уже существует"
- В этом случае просто войдите с этими данными
- Если не помните пароль, используйте функцию восстановления пароля

## Если всё ещё не работает

1. **Убедитесь, что отключили подтверждение email** (см. `DISABLE_EMAIL_CONFIRMATION.md`)
2. **Проверьте, что пользователь создан и подтвержден** в Supabase Dashboard → Authentication → Users
3. **Очистите кэш приложения** и попробуйте снова
4. **Попробуйте зарегистрироваться с новым email**, если этот уже занят
